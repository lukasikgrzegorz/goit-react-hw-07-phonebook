#  Phonebook 

Phonebook APP using mockAPI.

https://lukasikgrzegorz.github.io/goit-react-hw-07-phonebook

## 🚀 Run Locally

Clone the project

```bash
  git clone https://github.com/lukasikgrzegorz/goit-react-hw-07-phonebook
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

## 📝 Tech Stack

React, Redux
